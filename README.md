## Tentang Project

Aplikasi untuk pengelolaan proses klinik

## Cara Install
1. git clone https://github.com/nurisakbar/app-klinik.git
2. cd app-klinik
3. composer install
4. cp .env-example dan sesuaikan konfigurasi database
5. php artisan migrate --seed
6. php artisan serve
